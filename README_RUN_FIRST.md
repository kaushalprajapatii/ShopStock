# ShopStock Fixed Setup

## Run Instructions

1. Open terminal in project folder

2. Create virtual environment

Windows:
python -m venv venv
venv\Scripts\activate

Mac/Linux:
python3 -m venv venv
source venv/bin/activate

3. Install dependencies

pip install -r backend/requirements.txt

4. Start server

python run_local.py

5. Open browser

http://127.0.0.1:8000
